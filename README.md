> 基于 OpenClaw 架构经验，使用 Node.js/TypeScript 构建的轻量级 AI 助手网关。
> 
> 飞书接入（WebSocket 长连接）→ Qwen LLM（预留多模型扩展）→ 工具执行 → 飞书回复。
> 
> 目标环境：2核4G 云服务器，Docker 部署。
> 
> 会话管理，历史对话压缩，更轻量，更新节省tokens
> 
> 实现功能：通过飞书自然语言进行设置定时任务、自动采集新闻资讯
